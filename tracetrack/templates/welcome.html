Welcome to TraceTrack

TraceTrack is a web application for batch alignment of trace files to a reference and visualization of mutations. The tool enables batch upload of trace files, creating alignments with user-defined settings, displaying their overview of details and viewing original chromatograms

The input consists of trace files (in ab1 format) and a file with reference sequences (as .xlsx, .csv, GenBank or fasta format). Alignments of trace files to the reference are performed based on user settings and displayed in a table. After clicking on an alignment, the chromatogram from the ab1 file can be displayed. For more information, see the help tab.
